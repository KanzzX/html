Latihan HTML

HTML (HyperText Markup Language) adalah bahasa markup untuk membuat dan menyusun halaman web. HTML membentuk struktur dasar konten seperti teks, gambar, dan tautan. Versi terbarunya, **HTML5**, menambahkan fitur multimedia dan elemen semantik. HTML bekerja bersama CSS (untuk tampilan) dan JavaScript (untuk interaktivitas) dalam pengembangan web.
